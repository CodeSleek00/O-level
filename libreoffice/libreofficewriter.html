LibreOffice Writer Shortcut Keys

General Shortcuts

Shortcut Action
Ctrl + N Create a new document
Ctrl + O Open an existing document
Ctrl + S Save the current document
Ctrl + Shift + S Save As (Save with a new name)
Ctrl + P Print the document
F1 Open Help
F5 Open Navigator (for quick navigation)
F7 Open Spellcheck
F11 Open Styles and Formatting panel

Editing Shortcuts

Shortcut Action
Ctrl + C Copy selected text
Ctrl + X Cut selected text
Ctrl + V Paste clipboard content
Ctrl + Z Undo last action
Ctrl + Y Redo last undone action
Ctrl + F Find text
Ctrl + H Find and Replace
Ctrl + B Bold text
Ctrl + I Italicize text

Navigation Shortcuts

Shortcut Action
Home Move cursor to the start of the line
End Move cursor to the end of the line
Ctrl + Home Move cursor to the start of the document
Ctrl + Left/Right Arrow Move cursor word by word

Table Shortcuts

Shortcut Action
Ctrl + F12 Insert a new table
Tab (inside table) Move to the next cell
Alt + Arrow Keys Resize table columns/rows

Document Management

Shortcut Action
Ctrl + Enter Insert a Page Break
Ctrl + 1 Single line spacing
Ctrl + 2 Double line spacing

Special Characters & Fields

Shortcut Action
Ctrl + F2 Insert Fields (Date, Time, etc.)

View Shortcuts

Shortcut Action
Ctrl + Mouse Wheel Zoom in/out
